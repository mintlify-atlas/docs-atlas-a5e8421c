## Installation

```bash
npm install @jeremiemeunier/core
```

## Package Information

- **Version**: 1.4.7
- **License**: ISC
- **Repository**: [github.com/jeremiemeunier/windmill-components](https://github.com/jeremiemeunier/windmill-components)

## Components

### CircleGauge

A circular gauge component for displaying progress.

**Props:**

```typescript
interface CircleGaugeTypes {
  progress: number;
}
```

**Usage:**

```tsx
import { CircleGauge } from '@jeremiemeunier/core';

<CircleGauge progress={75} />
```

### Loader

A loading spinner component.

**Props:**

```typescript
interface LoaderProps {
  width?: number;
  color?: string;
}
```

**Usage:**

```tsx
import { Loader } from '@jeremiemeunier/core';

<Loader width={40} color="#3b82f6" />
```
