## Components

### Carousel

A responsive carousel component for displaying multiple slides.

**Props:**

```typescript
interface CarouselTypes {
  slides: React.ReactNode[];
  slidesNumber?: number;
  className?: string;
}
```

**Parameters:**

- `slides` (required): Array of React nodes to display as slides
- `slidesNumber` (optional): Number of slides to show at once
- `className` (optional): Additional CSS class names

**Usage:**

```tsx
import { Carousel } from '@jeremiemeunier/carousel';

const slides = [
  <div>Slide 1</div>,
  <div>Slide 2</div>,
  <div>Slide 3</div>,
];

<Carousel slides={slides} slidesNumber={1} />
```
