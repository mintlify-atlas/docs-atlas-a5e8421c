## Components

### CustomCursor

A custom cursor component that creates an animated cursor effect, framing links and interactive elements on hover.

**Props:**

```typescript
type Smoothness = {
  movement?: number;
  resize?: number;
};

type CustomCursorProps = {
  targets?: string;
  padding?: number;
  borderWidth?: number;
  borderColor?: string;
  radius?: number;
  idleSize?: number;
  smoothness?: Smoothness;
  zIndex?: number;
  disableOnTouch?: boolean;
  className?: string;
};
```

**Parameters:**

- `targets` (optional): CSS selector for target elements
- `padding` (optional): Padding around target elements
- `borderWidth` (optional): Width of the cursor border
- `borderColor` (optional): Color of the cursor border
- `radius` (optional): Border radius of the cursor
- `idleSize` (optional): Size of the cursor when idle
- `smoothness` (optional): Animation smoothness configuration
  - `movement`: Smoothness of cursor movement
  - `resize`: Smoothness of cursor resize animation
- `zIndex` (optional): Z-index of the cursor
- `disableOnTouch` (optional): Disable cursor on touch devices
- `className` (optional): Additional CSS class names

**Usage:**

```tsx
import { CustomCursor } from '@jeremiemeunier/cursor';

<CustomCursor
  targets="a, button"
  padding={10}
  borderWidth={2}
  borderColor="#3b82f6"
  radius={8}
  idleSize={8}
  smoothness={{ movement: 0.3, resize: 0.3 }}
/>
```
