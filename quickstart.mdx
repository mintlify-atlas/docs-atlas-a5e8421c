## Overview

This guide will walk you through creating a simple login form using Windmill Components. You'll learn the basic patterns and concepts that apply to all components in the library.

<Note>
  Make sure you've completed the [Installation](/installation) steps before proceeding.
</Note>

## Building a Login Form

Let's build a complete login form with email and password inputs, validation, and submission handling.

<Steps>
  <Step title="Install Required Packages">
    First, install the form components:

    <CodeGroup>
    ```bash npm
    npm install @jeremiemeunier/core @jeremiemeunier/form
    ```

    ```bash yarn
    yarn add @jeremiemeunier/core @jeremiemeunier/form
    ```

    ```bash pnpm
    pnpm add @jeremiemeunier/core @jeremiemeunier/form
    ```
    </CodeGroup>
  </Step>

  <Step title="Import Components">
    Import the necessary components from the form package:

    ```tsx
    import { Form, Input, Password, Submit } from "@jeremiemeunier/form";
    import { useState } from "react";
    ```
  </Step>

  <Step title="Create the Component">
    Set up your component with state management:

    ```tsx
    function LoginForm() {
      // State for form content
      const [content, setContent] = useState({
        email: "",
        password: ""
      });

      // State for validation errors
      const [error, setError] = useState({
        email: "",
        password: ""
      });

      // Form submission handler
      const handleSubmit = (event) => {
        event.preventDefault();
        console.log("Logging in with:", content);
        // Add your authentication logic here
      };

      return (
        // Form component will be added here
      );
    }
    ```
  </Step>

  <Step title="Build the Form">
    Add the Form component with input fields:

    ```tsx
    return (
      <Form
        content={content}
        setContent={setContent}
        error={error}
        setError={setError}
        submit={{
          label: "Sign In",
          action: handleSubmit,
          appearance: "primary"
        }}
      >
        <Input
          label="Email Address"
          content={content.email}
          setContent={(value) => setContent({ ...content, email: value })}
          error={error.email}
          type="email"
          placeHolder="Enter your email"
          required={true}
          name="email"
        />

        <Password
          label="Password"
          content={content.password}
          setContent={(value) => setContent({ ...content, password: value })}
          error={error.password}
          placeHolder="Enter your password"
          required={true}
          name="password"
        />
      </Form>
    );
    ```
  </Step>
</Steps>

## Complete Example

Here's the complete code for the login form:

```tsx LoginForm.tsx
import { Form, Input, Password, Submit } from "@jeremiemeunier/form";
import { useState } from "react";

function LoginForm() {
  const [content, setContent] = useState({
    email: "",
    password: ""
  });

  const [error, setError] = useState({
    email: "",
    password: ""
  });

  const handleSubmit = (event) => {
    event.preventDefault();
    
    // Basic validation
    const newErrors = { email: "", password: "" };
    
    if (!content.email) {
      newErrors.email = "Email is required";
    }
    
    if (!content.password) {
      newErrors.password = "Password is required";
    }
    
    if (newErrors.email || newErrors.password) {
      setError(newErrors);
      return;
    }
    
    // Submit the form
    console.log("Logging in with:", content);
    // Add your authentication logic here
  };

  return (
    <div style={{ maxWidth: "400px", margin: "0 auto", padding: "2rem" }}>
      <h1>Welcome Back</h1>
      <Form
        content={content}
        setContent={setContent}
        error={error}
        setError={setError}
        submit={{
          label: "Sign In",
          action: handleSubmit,
          appearance: "primary"
        }}
      >
        <Input
          label="Email Address"
          content={content.email}
          setContent={(value) => setContent({ ...content, email: value })}
          error={error.email}
          type="email"
          placeHolder="Enter your email"
          required={true}
          name="email"
        />

        <Password
          label="Password"
          content={content.password}
          setContent={(value) => setContent({ ...content, password: value })}
          error={error.password}
          placeHolder="Enter your password"
          required={true}
          name="password"
          tagline={{
            link: "/forgot-password",
            label: "Forgot password?"
          }}
        />
      </Form>
    </div>
  );
}

export default LoginForm;
```

## Understanding the Pattern

All Windmill form components follow a consistent pattern:

### State Management

<CodeGroup>
```tsx Content State
// Stores the actual values
const [content, setContent] = useState({
  fieldName: "initial value"
});
```

```tsx Error State
// Stores validation error messages
const [error, setError] = useState({
  fieldName: ""
});
```
</CodeGroup>

### Component Props

Every form component accepts these common props:

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `label` | string | optional | Label displayed above the input |
| `content` | any | required | Current value of the field |
| `setContent` | function | required | Function to update the value |
| `error` | string | optional | Error message to display |
| `required` | boolean | optional | Whether the field is required |
| `name` | string | optional | Name attribute for the input |
| `size` | number | optional | Size/width of the component |
| `readOnly` | boolean | optional | Makes the field read-only |
| `locked` | boolean | optional | Disables the field |
| `tagline` | object | optional | Additional link/text below the field |

## More Examples

### Using Checkboxes

```tsx
import { Checkbox } from "@jeremiemeunier/form";

const [agreed, setAgreed] = useState(false);

<Checkbox
  label="I agree to the terms and conditions"
  content={agreed}
  setContent={setAgreed}
  required={true}
  tagline={{
    link: "/terms",
    label: "Read our terms"
  }}
/>
```

### Using Select

```tsx
import { Select } from "@jeremiemeunier/form";

const [country, setCountry] = useState("");
const [error, setError] = useState("");

<Select
  label="Country"
  content={country}
  setContent={setCountry}
  error={error}
  required={true}
>
  <option value="">Select a country</option>
  <option value="US">United States</option>
  <option value="CA">Canada</option>
  <option value="UK">United Kingdom</option>
  <option value="FR">France</option>
</Select>
```

### Using DatePicker

```tsx
import { DatePicker } from "@jeremiemeunier/form";

const [birthDate, setBirthDate] = useState(new Date());
const [error, setError] = useState("");

<DatePicker
  label="Date of Birth"
  content={birthDate}
  setContent={setBirthDate}
  error={error}
  required={true}
  blockedDate={{
    type: "array",
    values: ["2024-12-25", "2024-01-01"] // Block holidays
  }}
/>
```

### Using TextArea

```tsx
import { TextArea } from "@jeremiemeunier/form";

const [message, setMessage] = useState("");
const [error, setError] = useState("");

<TextArea
  label="Your Message"
  content={message}
  setContent={setMessage}
  error={error}
  placeHolder="Enter your message here..."
  maxLength={500}
  rows={5}
  required={true}
/>
```

## Adding Toast Notifications

Enhance your form with toast notifications for user feedback:

<Steps>
  <Step title="Install Toast Package">
    ```bash
    npm install @jeremiemeunier/toast
    ```
  </Step>

  <Step title="Wrap Your App">
    ```tsx App.tsx
    import { ToastProvider } from "@jeremiemeunier/toast";

    function App() {
      return (
        <ToastProvider>
          <LoginForm />
        </ToastProvider>
      );
    }
    ```
  </Step>

  <Step title="Use in Your Form">
    ```tsx
    import { useToasts } from "@jeremiemeunier/toast";

    function LoginForm() {
      const { pushToast } = useToasts();

      const handleSubmit = (event) => {
        event.preventDefault();
        
        // Show success toast
        pushToast({
          title: "Success",
          content: "Login successful!",
          type: "positive",
          duration: 3
        });
      };

      // ... rest of the component
    }
    ```
  </Step>
</Steps>
