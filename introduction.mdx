## Welcome to Windmill Components

Windmill Components is a comprehensive collection of React libraries published under the `@jeremiemeunier` scope. Each package is independently published and maintained, allowing you to install only the components you need.

## Available Packages

| Package | Description |
| ------- | ----------- |
| **core** | Base components and utilities (e.g., `Loader`) |
| **carousel** | Carousel component for scrolling content |
| **cursor** | Custom cursor that highlights links |
| **drawer** | Sliding side panel component |
| **editorparser** | Parser to display JSON editor content |
| **form** | Large collection of form elements: `Input`, `Select`, `Checkbox`, `DatePicker`, etc. |
| **x-form** | FormData fork - form components optimized for automatic FormData extraction |
| **hooks** | Reusable React hooks like `useRefresh` and `useSeo` |
| **modal** | Complete modal system with many sub-components |
| **navigation** | Provider and hook for managing navigation state |
| **theme** | Theme management with context and selection components |
| **toast** | Toast notification library |
| **popover** | Popover component for contextual overlays |

## Quick Example

Here's a quick example of using the Form components:

```tsx
import { Form, Input, Submit } from "@jeremiemeunier/form";
import { useState } from "react";

function MyForm() {
  const [content, setContent] = useState({ email: "" });
  const [error, setError] = useState({ email: "" });

  const handleSubmit = (event) => {
    event.preventDefault();
    console.log("Form submitted:", content);
  };

  return (
    <Form
      content={content}
      setContent={setContent}
      error={error}
      setError={setError}
      submit={{
        label: "Submit",
        action: handleSubmit,
        appearance: "primary"
      }}
    >
      <Input
        label="Email Address"
        content={content.email}
        setContent={(value) => setContent({ ...content, email: value })}
        error={error.email}
        type="email"
        required={true}
      />
    </Form>
  );
}
```
